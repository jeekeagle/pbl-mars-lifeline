# Mars Lifeline PBL Course

## 火星生命线

A PBL course where students design a life support system for the first human settlement on Mars.

**Driving Question:** What are the most critical life support system challenges we must solve to establish the first human settlement on Mars?

## Course Structure

- **Duration:** 240 minutes
- **Phases:** 5
- **Final Deliverable:** Complete Mars Settlement System Integration Plan
